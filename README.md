# E-Commerce Shop

Full-stack online shop with React frontend and Node.js backend. Products, cart, checkout, order history.

> React 18 · Node.js · Express · SQLite · JWT · React Router

## Features
- Product catalog with search and category filter
- Shopping cart (add, update quantity, remove)
- JWT authentication (register/login)
- Checkout creates order and reduces stock
- Order history per user
- 6 sample products seeded on first run

## Quick start

```bash
# Backend
cd backend
copy .env.example .env
npm install && npm run dev

# Frontend
cd frontend
npm install && npm start
```

Frontend: http://localhost:3000
